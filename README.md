# pavilion-core

Metadata manager for anime, books, comics, games and music.

## Book

Download from these APIs:
- Google Book API
- ISBNDB API

Notice: Please specify ISBNDB API KEY in variable: `export ISBNDB_API_KEY=YOUR_API_KEY`.

## Music

Download from these APIs:
- MusicBrainZ API
